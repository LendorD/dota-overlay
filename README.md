# Dota Overlay

Краткий оверлей для Dota 2: показывает пики врагов, контр?пики по герою и лучшие общие пики.

## Возможности
- Чтение `console.log` и обнаружение выбранных героев.
- Контр?пики по конкретному герою (OpenDota).
- Лучшие общие пики против всех найденных врагов.
- Перетаскивание окна (в режиме unlock).
- Масштабирование окна `+` / `-`.

## Управление
- `F12` — Lock/Unlock (когда unlocked, окно кликабельно и перетаскивается).
- ЛКМ (клик) — переключение контр?пиков по найденным героям.
- `+` / `-` — масштаб окна.

## Запуск

1) Убедись, что установлен Go.
2) Запусти приложение:

```bash
go run ./cmd/overlay
```

## OpenDota API

Если есть ключ, можно передать через переменную окружения:

```powershell
$env:OPENDOTA_API_KEY="YOUR_KEY"
```

## Примечания
- В игре против ботов лог может не содержать всех пиков противников.
- Для полного списка героев приложение загружает `/heroes` из OpenDota при старте.
